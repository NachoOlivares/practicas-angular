This is a Spring Boot Project Application, so in order to use it you'll have to execute it as a Spring Boot app.
There is a jar file named sudokuService.jar in target which contains the microservice the app needs to run.
You can find this service in the folder src/main/app/com.sudokuSolve.service if you wanna check it.

Once you initiate the .jar service, it will run on port:8989.
The Spring Boot app will run on port:11111
